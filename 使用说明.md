# 文案管理系统使用说明

## 系统架构

### 文件结构
```
index.html         # 最终用户使用的文案展示页面
editor.html       # 编辑页面（编辑临时数据）
viewer.html       # 预览页面（预览和发布）
admin.html        # 旧版管理页面（暂时不用）
```

### 数据流
1. **临时数据** (`/data/xiaohongshu_temp.json`) - 编辑时保存
2. **正式数据** (`/data/xiaohongshu_data.json`) - 发布后生效，index.html读取
3. **编辑流程**：编辑 → 保存到临时 → 预览 → 发布到正式

## 使用方法

### 第一步：访问编辑页面
```
http://your-domain.com/editor.html
```

### 第二步：编辑文案
1. 点击"✏️ 编辑"按钮进入编辑模式
2. 输入框变为可编辑状态
3. 修改标题和文案内容
4. 点击"💾 保存"按钮保存到临时文件
   - 成功：切换回只读模式，显示保存成功
   - 失败：保持编辑状态，显示错误信息

### 第三步：预览文案
1. 点击"👁️ 去预览"按钮进入预览页面
2. 预览页面自动加载临时文件数据
3. 可以看到数据来源显示"临时文件"

### 第四步：发布文案
1. 在预览页面确认文案无误
2. 点击"🚀 发布"按钮将临时数据发布到正式环境
3. 发布成功后：
   - 数据来源变为"正式文件（已发布）"
   - 发布按钮变为禁用状态
   - index.html会立即读取新数据

### 第五步：用户查看
1. 用户访问 `index.html` 看到最新发布的文案
2. 可以点击"复制标题"/"复制文案"按钮使用

## Nginx配置要求

### WebDAV模块必须启用
```nginx
location /data/ {
    alias /mnt/mmc0-4/webdata/;   # 可写的存储目录
    dav_methods PUT DELETE MKCOL COPY MOVE;
    dav_access user:rw group:rw all:r;
    autoindex on;
    allow all;
    client_max_body_size 10M;
}
```

### 目录权限
```bash
# 创建存储目录
mkdir -p /mnt/mmc0-4/webdata/

# 设置权限（根据你的Nginx用户）
chown -R www-data:www-data /mnt/mmc0-4/webdata/
chmod -R 755 /mnt/mmc0-4/webdata/
```

## 文件说明

### 数据文件
1. **临时文件**：`/data/xiaohongshu_temp.json`
   - 编辑时保存到此文件
   - 预览页面读取此文件
   - 不影响最终用户

2. **正式文件**：`/data/xiaohongshu_data.json`
   - 发布时从此文件生成
   - index.html读取此文件
   - 最终用户看到的内容

### HTML文件
1. **index.html** - 面向最终用户
2. **editor.html** - 面向编辑人员
3. **viewer.html** - 面向审核人员
4. **admin.html** - 旧版（保留备用）

## 注意事项

1. **权限问题**：确保Nginx进程有存储目录的写权限
2. **临时文件**：编辑后请及时发布或清理临时文件
3. **并发编辑**：系统不处理并发编辑，请按顺序操作
4. **浏览器兼容**：现代浏览器支持良好，微信内置浏览器可能有PUT方法限制
5. **数据安全**：建议配置适当的访问控制

## 故障排除

### 常见问题
1. **无法保存**：检查Nginx WebDAV配置和目录权限
2. **无法读取**：检查文件路径和网络连接
3. **发布失败**：检查正式文件路径和权限
